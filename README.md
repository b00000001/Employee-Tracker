# Usage

![App Demo](employeetrackerdemo.gif)
In order to use this app, you will need to navigate to the menu you wish you use. From the main menu, you can go to a menu for Adding, Removing and Viewing employees. If you add an Employee, you will be asked a series of questions and when that is concluded, you can check the results in the view function. You can also dynamically remove entries from the database from the remove menu.
